# Vue 3 + Vite

Technologies Stack:
1. Vue 3
2. Vue Router
3. Pinia
4. Bootstrap-vue-3
5. Vite

Инструкция по запуску
1. Выполнить клонирование репозитория 
2. Перейти в cmd и перейти в каталог pay-divide 
3. Выполнить команду npm i, для установки всех нужных библиотек
4. выполнить команду npm run dev
